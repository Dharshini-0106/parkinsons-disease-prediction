## 📌 Overview
A machine learning model to predict Parkinson's disease using vocal 
biomarker data from the UCI Parkinson's dataset.

## 📊 Dataset
- **Source:** UCI Machine Learning Repository — Parkinson's Dataset
- **Features:** 22 vocal measurements including jitter, shimmer, and HNR
- **Target:** Parkinson's disease status (1 = Parkinson's, 0 = Healthy)

## ⚙️ Methodology
- ✅ Exploratory Data Analysis and correlation heatmap
- ✅ Train/Test split (80/20)
- ✅ Compared Logistic Regression, Random Forest, Decision Tree
- ✅ Evaluated using accuracy score, confusion matrix, and Cohen's Kappa
